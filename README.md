# Yolomy
This is an portfolio website for Yolomy Store.
### Getting Started
- Clone this repository:
    ```
    $ git clone https://github.com/HannesKimara/Yolomy.git
    ```
- cd into `Yolomy`:

    ```
    $ cd Yolomy 
    ```
### Usage
- Open `index.html`
- You could also visit the live demo [here](https://hanneskimara.github.io/Yolomy/)

### Built with
- HTML
- CSS
- JavaScript
- JQuery
- Bootstrap
